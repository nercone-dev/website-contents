# Nercone Credit
ここではnercone.devおよびNerconeによって制作された一部の著作物で使用されているプロジェクト・ソフトウェア・アセット等の一覧を確認できます。

一覧にはNercone自身によって制作された著作物は記載していません。

この一覧は完全ではなく、記載漏れがあることをご了承ください。

## フォント
パフォーマンス改善などの理由で、一部のフォントは複数のフォントを合成したものを使用している場合があります。

### Inter
ラテン文字用のサンセリフフォントです。
[Google Fonts](https://fonts.google.com/specimen/Inter/)から入手しました。

### MesloLGS Nerd Font
等幅フォントです。
[Nerd Fonts](https://nerdfonts.com/)から入手しました。

### BIZ UDGothic
日本語用のサンセリフフォントです。
[Google Fonts](https://fonts.google.com/specimen/BIZ+UDGothic/)から入手しました。

### Noto Sans JP
日本語用のサンセリフフォントです。
[Google Fonts](https://fonts.google.com/noto/specimen/Noto+Sans+JP/)から入手しました。

### Noto Sans SC
中国語(簡体字)用のサンセリフフォントです。
[Google Fonts](https://fonts.google.com/noto/specimen/Noto+Sans+SC/)から入手しました。

### Noto Sans TC
中国語(繁体字)用のサンセリフフォントです。
[Google Fonts](https://fonts.google.com/noto/specimen/Noto+Sans+TC/)から入手しました。

### Noto Sans KR
韓国語用のサンセリフフォントです。

## 画像

### ソーシャルアイコン
ソーシャルアイコンの使用方法は各配布元に記載しているガイドラインに準拠しています。

アイコンは画像編集ソフトでサイズや色を調整したものを使用しています。
編集後の色はnercone.devのデフォルトの`dark`テーマ時のデフォルトの背景色に対して十分なコントラストがあり、WCAG AAのコントラスト比4.5:1以上の要件に合格しています。

- **GitHub** [Brand Toolkit](https://brand.github.com/foundations/logo/)から入手しました。
- **ORCID** [ORCID Brand Library](https://orcid.filecamp.com/s/o/3CCuLloCl73Knntn/)から入手しました。
- **Qiita** [メディアキット](https://help.qiita.com/ja/articles/others-brand-guideline)から入手しました。
- **Twitter** 公式からの配布が終了していたため、[Wikimedia Commons](https://commons.wikimedia.org/wiki/File:Logo_of_Twitter.svg)から入手しました。
- **Zenn** [メディアキット](https://zenn.dev/mediakit/)から入手しました。

## ソフトウェア

### Nginx
TLSやQUICなどに対応するためにリバースプロキシとして使用しています。
[nginx: Linux packages](https://nginx.org/en/linux_packages.html#RHEL)に記載されている手順でパッケージリポジトリを追加しインストールしました。

### uv
Pythonのインストール・バージョン管理やパッケージの管理・ビルドに使用しています。
[Installing uv](https://docs.astral.sh/uv/getting-started/installation/)に記載されている手順でインストールしました。

### Python
メインで使用しているプログラミング言語です。
ほとんどの場合、uvでCPython 3.12.xをインストールし使用しています。
